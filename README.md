# Task Manager

A full-stack task management application built with a Java + Spring Boot backend and a React + Vite frontend, supporting:

* Secure user registration and login with JWT 🔐
* Full CRUD functionality for creating, updating, and managing tasks and lists ✅
* User-specific data access ensuring each user only sees their own content 🛡️
* A responsive, modern design with Tailwind CSS 📱
* Global state management with Zustand for a reactive and instant UI ⚡
* A clean, modern UI communicating with a custom-built REST API

---

## 🛠️ Tech Stack

### Backend

* **Java & Spring Boot:** For building the robust and secure REST API.
* **Spring Security:** For handling JWT-based authentication and authorization.
* **PostgreSQL & Docker:** For a persistent and containerized database.
* **JPA & Hibernate:** For object-relational mapping.
* **Flyway:** For version-controlled database migrations.

### Frontend

* **React & TypeScript:** For building a type-safe and component-based user interface.
* **Vite:** For a fast development server and optimized bundling.
* **Tailwind CSS:** For a utility-first styling approach.
* **Zustand:** For simple and powerful global state management.
* **Axios:** For making HTTP requests to the backend API.
